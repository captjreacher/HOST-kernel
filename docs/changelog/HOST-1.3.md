# HOST-1.3 Taxonomy Resolver

## Summary

Implements the canonical runtime taxonomy resolver for the HOST Core Kernel.

## Included

- Canonical object type lookup
- Identifier prefix lookup
- Lifecycle state lookup
- Event type lookup
- Relationship type lookup
- Validation and discovery

## Not Included

- Persistence
- Runtime mutation
- Product-specific taxonomy

## Validation

- `npm run build`
- `npm test`
- `npm run verify:graph`
